# HeliBoard Layouts

[HeliBoard](https://github.com/Helium314/HeliBoard) is an open-source Android keyboard.

## Layouts

- `normal/` — Standard QWERTY layout
- `termux/` — PC-style layout

## Usage

Copy the `.jsonc` files to your device and import them in HeliBoard:
**Settings → Layouts → Custom layouts → Add layout**
